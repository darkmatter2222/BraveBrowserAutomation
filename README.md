# BraveBrowserAutomation  
This repo [Youtube Tutorial](https://youtu.be/N3Nqq4Rl9sE)  

## Objective  
 - Emulate Page Navigation
 - Emulate Open/Closing Tabs
 - Emulate UI Element Clicking
 - HID (Human Interface Device) Simulation
